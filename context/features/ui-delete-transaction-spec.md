# UI Delete Transaction Spec

## Overview

Add the ability to remove a transaction from the budget tracking dashboard.

## Requirements

- Add a button add the end of the row to delete the transaction.
- Button should only appear when hovering on row.
- Button should be a red X.
- A confirmation modal should pop up to confirm deletion.
- Removing a transaction deletes the row from the database.
